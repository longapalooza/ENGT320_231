08.Example 1¶

Given:¶

Experimental results for the local heat transfer coefficient $h_x$ for flow over a flat plate with an extremely rough surface were found to fit the relation
    $h_x\left(x\right) = a x^{-0.1}$
where $a$ is a coefficient $\left(\frac{W}{m^{1.9}K}\right)$ and $x \left(m\right)$ is the distance from the leading edge of the plate.

Required:¶

Develop an expression for the ratio of the average heat transfer coefficient $\bar{h}_x$ for a plate of length $x$ to the local heat transfer coefficient $h_x$ at $x$. Plot the variation of $h_x$ and $\bar{h}_x$ as a function of $x$.

Solution:¶

Start first by importing any needed libraries.

In [1]:
import matplotlib.pyplot as plt
import numpy as np
import pint # https://pint.readthedocs.io

The unit registry may then be defined. This is shown below.

In [2]:
ureg=pint.UnitRegistry()
ureg.setup_matplotlib()
Q_=ureg.Quantity

Beginning with the expression for $\bar{h}_x$ for a flat plate,
    $\bar{h}_x\left(x\right) = \frac{1}{x} \int_0^x h_x\left(x\right) dx$
Entering the given expression for the local convective coefficient shows
    $\bar{h}_x\left(x\right) = \frac{1}{x} \int_0^x a x^{-0.1} dx$
Since the $a$ term is constant, it may be taken out of the integral as shown by
    $\bar{h}_x\left(x\right) = \frac{a}{x} \int_0^x x^{-0.1} dx$
At this point integration may be performed. This is shown below.
    $\bar{h}_x\left(x\right) = \frac{a}{x} \left( \frac{x^{0.9}}{0.9}\right)\biggr \rvert_{0}^{x}= \frac{a}{x} \left( \frac{x^{0.9}}{0.9}\right)$
Simplifying shows
    $\bar{h}_x\left(x\right) = 1.\bar{1} a x^{-0.1}$

Referring to the given expression for the local convective coefficient $h_x$ shows that the average convective coefficient may be expressed as
    $\bar{h}_x\left(x\right) = 1.\bar{1} h_x\left(x\right)$

Defining a Python function will be a useful way to plot these functions. A Python function for both $h_x$ and $\bar{h}_x$ are defined below. Note the $a$ term is omitted. This is intentional. The $a$ is simply a scaling term and the ordinate axis ($y$ axis) will be put in terms of $a$ so that a single plot may be created to represent any $a$ value.

In [3]:
def h_xFunc(x):
    return x**(-0.1)

def hb_xFunc(x):
    return (1/0.9)*h_xFunc(x)

The plotting values for $x$ and the $h$ terms will be created using Numpy's linspace. This is shown below.

In [4]:
x=np.linspace(0.1, 4, 100)*ureg.m

h_x=h_xFunc(x)*(ureg.W/(ureg.m**(1.9)*ureg.K))
hb_x=hb_xFunc(x)*(ureg.W/(ureg.m**(1.9)*ureg.K))

As of this writing, it appears that there is a bug in the Pint library that causes an error when trying to plot with units. See here for status of bug. To avoid this, the units will be removed and the unitless values will be plotted. Removing the units is shown below.

In [5]:
x=x.to('m').magnitude
h_x=h_x.to('W/(m**2*K)').magnitude
hb_x=hb_x.to('W/(m**2*K)').magnitude

The plotting handle may now be created and plotting elements added to the plot. This is shown below.

In [6]:
fig, ax=plt.subplots(1, 1)

ax.plot(x, h_x, color='k', label='$h_x$')
ax.plot(x, hb_x, color='k', linestyle='dashed', label='$\\bar{h}_x$')

ax.set_xlabel('x $\\left(m\\right)$')
ax.set_ylabel('$h\\ \\left( \\frac{W}{m^2 K}\\right)$')

ax.set_yticks([0, 0.5, 1, 1.5])
ax.set_yticklabels(['', '$0.5a$', '$a$', '$1.5a$'])

ax.legend()

plt.show()
