09.Example 1

Given:

Air at a pressure of $6 kPa$ and a temperature of $300 ^\circ C$ flows with a velocity of $10 \frac{m}{s}$ over a flat plate $0.5 m$ long.

Required:

Estimate the cooling rate per unit width of the plate needed to maintain it at a surface temperature of $27 ^\circ C$.

Solution:

Start first by importing any needed libraries.

In[1]:
import pint # https://pint.readthedocs.io

The unit registry may then be defined. This is shown below.

In[2]:
ureg=pint.UnitRegistry()
Q_=ureg.Quantity

The given problem variables may now be defined with units.

In[3]:
P=Q_(6, 'kPa')
T_inf=Q_(300, 'degC')
u_inf=Q_(10, 'm/s')
L=Q_(0.5, 'm')
T_s=Q_(27, 'degC')

There will be several material properties that need to be interpolated so a Python function will be defined to perform the interpolation function. This is shown below.

In[4]:
def interp(x, x1, x2, y1, y2):
    return ((x-x1)/(x2-x1))*(y2-y1)+y1

The film temperature is given
$T_f = \frac{T_\infty + T_s}{2}$

In[5]:
T_f=(T_inf.to('K')+T_s.to('K'))/2
round(T_f.to('K'), 1)
Out[5]:
436.6 kelvin

Going to Table A.4 @ $T_f$ shows that interpolation is needed. This is shown below.

In[6]:
Ta=Q_(400, 'K')
Tb=Q_(450, 'K')
nua=Q_(26.41E-6, 'm**2/s')
nub=Q_(32.39E-6, 'm**2/s')
nu_1=interp(T_f, Ta, Tb, nua, nub)
'{:~.3E}'.format(nu_1)
Out[6]:
'3.079E-05 m ** 2 / s'
In[7]:
ka=Q_(33.8E-3, 'W/(m*K)')
kb=Q_(37.3E-3, 'W/(m*K)')
k=interp(T_f, Ta, Tb, ka, kb)
'{:~.3E}'.format(k)
Out[7]:
'3.637E-02 W / K / m'
In[8]:
Pra=0.69
Prb=0.686
Pr=interp(T_f, Ta, Tb, Pra, Prb)
round(Pr, 4)
Out[8]:
0.6871 dimensionless

Table A.4 values are for atmospheric pressure (i.e. $1 atm$). However, the actually pressure is at $P=6 kPa$. It is known that the thermal conductivity $k$ and Prandtl number $Pr$ are independent of pressure. However, the kinematic viscosity $\nu$ is not independent on pressure. Previously, it was shown that the kinematic viscosity is inversely related to the pressure. That is
$\frac{\nu_2}{\nu_1}=\frac{P_1}{P_2}$
For this particular case, this becomes
$\nu=\nu_1\frac{P_{atm}}{P}$

In[9]:
nu=nu_1*Q_(1, 'atm')/P
'{:~.3E}'.format(nu.to('m**2/s'))
Out[9]:
'5.200E-04 m ** 2 / s'

The Reynolds number is then
$Re_L=\frac{u_\infty L}{\nu}$

In[10]:
Re_L=u_inf*L/nu
round(Re_L.to(''), 0)
Out[10]:
9615.0 dimensionless

Since $Re_L \lt Re_{x,c} = 5 \times 10^{-5}$, the flow is laminar. For laminar flow across a flat plate, the following empirical relation is valid.
$\bar{Nu}_L=0.664 Re_L^{1/2} Pr^{1/3}$

In[11]:
Nub_L=0.664*Re_L**(1/2)*Pr**(1/3)
round(Nub_L.to(''), 2)
Out[11]:
57.45 dimensionless

The definition for the Nusselt number may be used to find the average convective coefficient. This is shown below.
$\bar{Nu}_L=\frac{\bar{h} L}{k}$
Solving for the average convective coefficient shows
$\bar{h}=\frac{\bar{Nu}_L k}{L}$

In[12]:
hb=Nub_L*k/L
round(hb.to('W/(m**2*K)'), 3)
Out[12]:
4.179 watt/(kelvin meter2)

The required cooling rate per unit width is given by
$q^{'}=\bar{h}L \left( T_\infty - T_s\right)$

In[13]:
qp=hb*L*(T_inf-T_s)
round(qp.to('W/m'), 1)    # answer
Out[13]:
570.4 watt/meter
